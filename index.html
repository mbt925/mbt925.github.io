A handy Kotlin coroutine extension for switching dispatcher

mbt925 published on 2024-03-02
Imagine you have a suspend function like this: suspend fun doSomething() = withContext(Dispatchers.IO) { // body } This looks alright. Using withContext every time looks a bit verbose. There’s a nifty Kotlin extension on the invoke function that simplifies the code and makes it cleaner. This is a tip A tip banner 1 2 3 4 5 import kotlinx.coroutines.invoke suspend fun doSomething() = Dispatchers.IO { // body } The invoke extension allows you to specify the dispatcher directly within the function call, eliminating the need for withContext.
